# seconde-hand-market-api
중고거래 플랫폼의 백엔드 REST API 서버입니다.

## 🛠 Tech Stack 
- **Runtime**:Node.js
- **Framework**:Express.js
- **Database**:MariaDB
- **Storage**: AWS S3(이미지 업로드)
- **Auth**: JWT

## 📌 주요 기능 
- 회원가입/ 로그인 (JWT 인증)
- 상품 등록/ 조회/ 수정/ 삭제
- 이미지 업로드 (AWS S3)
- 채팅 (구매자 ↔ 판매자)
- 찜하기 / 거래 상태 관리
